# newsclip-news-snapshot

Fetch a page and create a hashed snapshot record for diffing later.

Tiny tool that stays out of the way and does one job.

## Why this exists
- Provide a simple starting point that can be expanded into a fuller tool.
- Provide a simple starting point that can be expanded into a fuller tool.

## Useful pages
- [Newsclip](https://newsclip.com/)
- [Newsclip Sports](https://newsclip.com/sports/)
- [Newsclip Terms](https://newsclip.com/terms/)

## Usage
Try it like this:

```bash
npm install
npm run start -- https://newsclip.com/sports/
```

## Notes
- Best-effort parsing is used to keep dependencies minimal.
- Designed to be safe to run in CI or cron contexts.
